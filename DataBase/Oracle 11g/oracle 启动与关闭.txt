登陆步骤

1 切换用户，并刷新变量
[oracle@oracle11g root]$ su  oracle;
[oracle@oracle11g root]$ source ~/.bash_profile

2 修改oratab
[oracle@oracle11g ~]$ vi /etc/oratab 

选择要启动的数据库实例名
hello:/home/work/oracle/product/11.2.0/db_1:Y
mail:/home/work/oracle/product/11.2.0/db_1:N

3 声明ORACLE_SID实例名
[oracle@oracle11g root]$ echo $ORACLE_SID
mail
[oracle@oracle11g root]$ export ORACLE_SID=hello;

4 启动oralce数据库
[oracle@oracle11g root]$ dbstart
Processing Database instance "hello": log file /home/work/oracle/product/11.2.0/db_1/startup.log
