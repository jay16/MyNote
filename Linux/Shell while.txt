bash的while loop:
Php代码  收藏代码

    语法:  
    while [ condition ]   
    do  
        command1  
        command1  
        commandN  
    done  
    示例:  
    #!/bin/bash  
    c=1  
    while [ $c -le 5 ]  
    do  
        echo "Welcone $c times"  
        (( c++ ))  
    done  



ksh的while loop:
Php代码  收藏代码

    语法:  
    while [[ condition ]] ; do  
        command1  
        command1  
        commandN  
    done  
      
    示例：  
    #!/bin/ksh  
    c=1  
    while [[ $c -le 5 ]]; do  
        echo "Welcome $c times"  
        (( c++ ))  
    done
